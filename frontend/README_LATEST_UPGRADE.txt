How to install:

2. Copy the lib folder and pubspec.yaml.
3. Paste them into:
   ai_exam_assistant/frontend
4. Replace existing files.
5. Run:
   flutter clean
   flutter pub get
   flutter run -d windows

For Android:
   flutter build apk --release
